# mini-project

:hourglass: 2023.04.03 ~ 2023 04.06

:bulb: 정글 입성 첫 프로젝트

## 프로젝트 소개
- 프로젝트명 : 정글인의 책상
- 기획의도
  - 같이 정글 생활을 할 동료들의 도구(키보드, 마우스 등) 정보 공유
  - 도구의 현실적인 사용후기 제공
- 기능
  - 코친진분들의 도구 정보 제공
  - 회원들의 정보 조회 및 검색 기능 제공
  - 회원들의 도구 선호도 시각화 자료 제공
- 트러블 슈팅
  - JWT 인증 방식을 활용하여 얻은 토큰을 가지고 user의 ID를 추출
  - 시각화 자료를 보여줄 때 값이 없는 예외상황이 존재하여 회원가입시 필수 입력으로 변경
